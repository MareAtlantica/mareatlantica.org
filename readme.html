Instalación: A famosa instalación en 5 minutos

  1. Descomprime o ficheiro nun cartafol baleiro e sube todo o seu contido.
  2. Abre wp-admin/install.php no teu navegador. Isto vai conducirte durante o proceso de configuración do ficheiro wp-config.php cos detalles da conexión á túa base de datos.
    1. Se por algunha razón isto non funcionara, non te preocupes. Non funciona en todos os servidores web. Abre wp-config-sample.php cun editor de texto como WordPad ou algún similar e introduce os detalles da conexión á túa base de datos.
    2. Garda o ficheiro como wp-config.php e súbeo.
    3. Abre wp-admin/install.php no teu navegador.
  3. Unha vez que o ficheiro de configuración está listo, o instalador configurará as táboas que o teu blogue necesita. Se houbera un erro, comproba unha vez máis o teu ficheiro wp-config.php, e téntao de novo. Se falla outra vez, por favor, vai aos foros de soporte con todos os datos que poidas aportar.
  4. Se non introduciches un contrasinal, apunta o contrasinal que se che dá. Se non proporcionaches un nome de usuario, este será admin.
  5. O instalador enviarache entón á then send you to the páxina de inicio de sesión. Inicie a sesión co nome de usuario e o contrasinal que escolliches durante a instalación. Se o contrasinal foi xenerado automaticamente, podes premer en 'Perfil' para cambiar o contrasinal.

Actualizando

Usando o actualizador automático

Se estás actualizando desde a versión 2.7 ou superior, podes usar o actualizador automático:

  1. Abre o ficheiro wp-admin/update-core.php no teu navegador e segue as instruccións.
  2. Querías máis, tal vez? Iso é todo!

Actualizando manualmente

  1. Antes de actualizar nada, asegúrate de ter copias de seguranza de todos os ficheiros que puideras ter modificado, como index.php.
  2. Borra os teus vellos ficheiros de WordPress, gardando aqueles que modificaras.
  3. Carga os novos ficheiros.
  4. Navega para /wp-admin/upgrade.php.

Cambios nos modelos dos temas

Se personalizaches os teus modelos de tema, terás que facer algúns cambios entre versións principais.

Migrando desde outros sistemas

WordPress pode importar desde determinado número de sistemas. Primeiro terás que ter WordPress instalado e traballando, tal e como describimos anteriormente, antes de usar as nosas ferramentas de importación.

Requirimentos do sistema

  • PHP versión 5.2.4 ou superior.
  • MySQL versión 5.0 ou superior.

Recomendacións do sistema

  • O módulo Apache mod_rewrite.
  • Unha ligazón a http://wordpress.org no teu sitio.

Recursos en liña

Se tes algunha dúbida que non abordamos neste documento, por favor, aproveita os numerosos recursos en liña de WordPress:

O Codex de WordPress
O Codex é a enciclopedia de todas as cousas WordPress. É a máis completa fonte de información dispoñible sobre WordPress.
O blogue de WordPress
Aquí é onde vas atopar as últimas actualizacións e novas relacionadas co WordPress. As noticias recentes de WordPress aparecen no panel administrativo, por defecto.
WordPress Planet
O WordPress Planet é un agregador de noticias que reúne artigos de blogues WordPress de toda a rede.
Foros de soporte de WordPress
Se xa buscaches en todas as partes e aínda non puideches atopar unha resposta, os foros de soporte son moi activos e teñen unha grande comunidade preparada para axudar. Para axudarlles a axudar, non te esquezas de empregar un título descritivo e describir a túa dúbida coa máxima información que che sexa posible achegar.
Canal IRC de WordPress
Existe unha canle de chat en liña que se usa para o debate entre as persoas que utilizan o WordPress e ocasionalmente para temas de soporte. A páxina wiki anterior debe levalo na dirección correcta. (irc.freenode.net #wordpress)

Entornos XML-RPC e Atom

Podes publicar no teu blog WordPress con ferramentas como o Windows Live Writer, Ecto, w.bloggar, Radio Userland (o que significa que pode utilizar o recurso de Radio email-to-blog), NewzCrawler, e outras ferramentas que soportan as APIs dos blogs! :) Podes ler máis sobre soporte XML-RPC no Codex.

Publicar vía correo electrónico

Podes publicar desde un cliente de correo electrónico! Para configuralo, vai á túa  pantalla de opcións de escritura e enche os detalles da conexión da túa conta POP3 secreta. Entón tes que configurar wp-mail.php para executar periodicamente a comprobación da bandexa de entrada na procura de novas publicacións. Podes facelo co cron-jobs, ou se o teu servidor non soporta iso, podes mirar na páxina web de seguimento de servizos diversos, e facer que comproben o URL do teu wp-mail.php.

Publicar é fácil: calquera correo enviado ao enderezo que especificaches será publicado, co asunto como título. É mellor manter esta dirección. O script borrará mensaxes que foran publicadas con éxito.

Notas finais

  • Se tes algunha suxestión, idea ou comentario, ou se (contra!) atopas un erro, únete a nós nos foros de soporte.
  • WordPress ten un robusto plugin API que fai o código fácil. Se és un desenvolvedor interesado en utilizar esta función, consulta a documentación sobre plugins no Codex. Non terás que modificar nada no código do núcleo.
